# sources/fb-hebrew/

Drop your raw Facebook Hebrew content here. The wiki ingests **from** this folder — never write here directly from `/wiki-*` commands.

Subfolders:
- `posts/` — one folder per shipped post, each containing `post-final.md` (the actual text) plus any pipeline drafts/briefs.
- `analytics/` — exports from Meta Business Suite (CSV). The wiki reads these to populate `avg_perf` on patterns/hooks/themes.

This is **immutable raw data**. Treat it like a read-only archive.
